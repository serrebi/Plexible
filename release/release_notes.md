## Features

### Complete Plex API Feature Coverage
This release adds comprehensive support for all Plex API features:

**Watchlist Management**
- Add/remove items from your Plex watchlist
- Check if items are on your watchlist
- List watchlist with filtering and sorting

**Playlist Management**
- Create, edit, and delete playlists
- Add/remove/reorder playlist items
- Copy playlists to other users
- Support for smart playlists

**Collection Management**
- Create, edit, and delete collections
- Add/remove/reorder collection items
- Support for smart collections

**Library Management**
- Trigger library scans and updates
- Cancel running scans
- Empty library trash
- Clean bundles and optimize database
- Refresh and analyze library sections

**Media Management**
- Remove items from Continue Watching
- Mark items as watched/unwatched
- Subtitle management (upload, search, download, remove)
- Create optimized versions of media
- Delete and refresh items

**User & Sharing Management**
- Invite and manage friends
- Create and manage home users
- Update sharing permissions
- Switch between users

**Server Administration**
- View and manage server settings
- Monitor active sessions and transcode sessions
- Run butler (maintenance) tasks
- Check for and install server updates
- View bandwidth statistics

**Discovery & History**
- Browse content hubs and recently added
- Search Plex Discover
- View watch history (server, section, account level)

**Additional Features**
- Webhook management
- Real-time alert listener for server events
- Sync features
- Client management
- Download media, databases, and logs

### Test Suite
Added comprehensive test suite with 147 tests covering all new features.
